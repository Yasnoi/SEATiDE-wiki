# 服务器简介

Seatide 服务器是一个永久周目的原版生存服务器，基于 Paper 服务端。Seatide 服务器倾向于提供
- 接近原版的生存体验
- 不设管理组、不设权限组的纯净游戏体验
- 永久主世界、下界存档
- 基础的红石机器支持

服务器基本概况：

- 开放时间：2026-01-23
- 游戏版本：1.21.11
- 服务端：Paper

游戏设置：
- 有正版验证
- 有白名单
- 没有死亡不掉落
- 仅苦力怕防爆
- sethome 开启，每人可至多设置 3 个传送点
- back 开启，支持死亡地点 back

平台设置：
- 每 5 分钟自动保存本地存档
- 每 10 分钟自动备份一次到 OSS
- 服务器空闲 20 分钟后自动删除

服务器没有专门的客户端，你可以使用自己喜欢的客户端进入游戏。
