# 加入/进入服务器

此页面介绍如何加入/进入 Seatide 服务器。

在加入/进入服务器之前，请先确保你有一个属于自己的正版 Minecraft 账号。

## 1. 申请白名单

到网页 https://v.wjx.cn/vm/m93QvcR.aspx （由问卷星提供）花五分钟填写白名单问卷。

## 2. 等待获取白名单
白名单审核一般在 24 小时以内完成。有以下两种方式可以获知你是否有白名单：
- 加入 QQ 讨论群（推荐）：可[加入我们的 QQ 讨论群](https://qm.qq.com/q/XijStp4T6u)，届时会直接在群内通知你。
- 如果你在填写问卷时包含了邮箱地址，会通过发邮件的形式通知你。

## 3. 准备客户端 & 加入服务器

获得白名单之后就可以进入服务器了。
- 准备客户端：可使用主流启动器，如 PCL、HMCL、Modrinth 或原版启动器作为客户端载体，以正版方式启动游戏
- 用 IP 或者域名加入服务器

我们提供两种服务器地址：
- IP 地址（稳定），你可以到[服务器状态页面](https://st.subilan.win/#/status)查看最新的 IP 地址，并使用游戏内「直接连接」选项进入服务器
- 域名（方便），在服务器正在运行的时候，你可以直接用域名 **mc.subilan.win** 连接。该域名可以使用的前提是 DNS 已经更新了域名与 IP 的解析关系，延迟大约在 1~5 分钟。

:::tip[动态 IP 地址]
服务器的 IP 地址会在每次开启时自动分配，因此服务器的 IP 地址是不断变化的，所以你需要前往服务器状态页面才能看到最新的准确 IP 地址。域名连接方式正是为了方便玩家无需查询 IP 地址而开通的。
:::
